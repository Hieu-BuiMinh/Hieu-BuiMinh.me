## Minimalist Resume 📚

Notion - [https://hieu-buiminh-resume.io.vn/](https://hieu-buiminh-resume.io.vn/)

Simple web app that renders minimalist CV with print-friendly layout.

Built with Next.js and shadcn/ui, deployed on Vercel.

<Image
	className="h-[410px] max-w-full m-auto"
	src="/assets/images/content/post/project-blog/minimalist-resume/cover-01.png"
/>

-   NextJS 14: https://nextjs.org/
-   Shadcn UI: https://ui.shadcn.com/

## Key Features

-   Light and Dark mode 🌓
-   Full mobile responsiveness 📱
-   Built using Next.js 15, React, Typescript, Shadcn/ui, TailwindCss 💄
-   Optimized for Next.js and Vercel 🚀
-   Print-friendly layout 🖨️
